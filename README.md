# Quantix Explorer

Etherscan-style block explorer for the Quantix post-quantum blockchain testnet.

**Live:** https://testnet.qpqb.org

## Features

- 📊 Dashboard with network stats
- 🧱 Block list and details
- 💸 Transaction list and details
- 👛 Address balances and history
- 🔍 Universal search

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- React Query

## Development

```bash
# Install
npm install

# Dev server
npm run dev

# Build
npm run build
```

## Environment Variables

```env
NEXT_PUBLIC_RPC_URL=https://rpc.qpqb.org
NEXT_PUBLIC_NETWORK=testnet
```
